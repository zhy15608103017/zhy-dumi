# [zhy-cli](https://github.com/zhy15608103017/zhy-dumi.git)

该脚手架模板为线上地址，更改线上仓库即可更改模板根据自己需求自动调整



### 使用说明

```jsx | pure
npm i zhydumi-cli
zhydumi-cli|npx zhydumi-cli
```
### 参数

foreendexport

| 参数            | 说明               | 
| --------------- | ------------------ |
| --help(-h)         | 帮助命令             |

|--install(-i)       | 是否自动下载依赖      |
